## 🌱 About The Project

India wastes nearly **40% of its food** every year — worth over ₹92,000 crore. Fresh Start India is a platform that tackles this crisis head-on by creating a smart redistribution network:

- 🥦 **Fresh surplus produce** from farmers and retailers is sold at lower prices to consumers
- 🌿 **Spoiled or unsellable items** are converted into affordable organic fertilizers
- 🔄 **Circular economy** — nothing goes to waste, everyone benefits

This project was built to explore how technology can solve real social and environmental problems at scale.

## 🏃 Getting Started

### Prerequisites
- Node.js v18+
- npm or bun

### Installation

```bash
# Clone the repo
git clone https://github.com/amithiprasad-cse/fresh-start-india.git

# Navigate into the project
cd fresh-start-india

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
